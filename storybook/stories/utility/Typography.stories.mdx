# Typography

## Font Weight

Utilities for controlling the font weight of an element. Quickly change the font weight of text with these utility classes.

-   `made-u-text-weight-bold`
-   `made-u-text-weight-medium`
-   `made-u-text-weight-light`
-   `made-u-text-weight-default`

## List Reset

The `made-u-list--reset` class when applied to the `ul` and `ol` elements will remove the default styles applied to lists.

`list-style` will be set to none and `margin-left` and `padding` will both be set to `0`.

<Canvas>
	<Story name="List">
		<ul class="made-u-list--reset">
			<li>List Item</li>
			<li>List Item</li>
			<li>List Item</li>
			<li>List Item</li>
			<li>List Item</li>
		</ul>
	</Story>
</Canvas>
