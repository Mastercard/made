# Flex

## Display Flex

Apply flex utility classes to create a flexbox container and transform direct children elements into flex items. Flex containers and items are able to be modified further with additional flex properties.

The classes are named using the format:

-   `made-u-display-flex`
-   `made-u-display-inline-flex`

Responsive variations also exist for `d-flex` and `d-inline-flex`.

-   `made-u-display-flex`
-   `made-u-display-inline-flex`
-   `made-u-display-flex--md`
-   `made-u-display-inline-flex--md`
-   `made-u-display-flex--lg`
-   `made-u-display-inline-flex--lg`

Responsive classes align with the medium and large breakpoints as outlined on the <a onClick={linkTo("foundations-grid--overview")}>Grid</a> page.

## Flex Direction

Set the direction of flex items in a flex container with direction utilities. In most cases you can omit the horizontal class here as the browser default is row. However, you may encounter situations where you needed to explicitly set this value (like responsive layouts).

Use `made-u-flex-row` to set a horizontal direction (the browser default), or `made-u-flex-row-reverse` to start the horizontal direction from the opposite side.

## Flex

Use this class to specify the ability of a flex item to alter its dimensions to fill available space.

### Flex 1

Adding `made-u-flex-1` makes the item grow in size to take up all the space that is available. Adding `made-u-flex-1` to all flex items results in each item having the same size.
