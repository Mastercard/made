# Space

We use the below defined tokens to create a predictable and harmonious spacing for our interface.
Global spacing tokens are based on multiples of 4px and should be used for all padding, margin and
position coordinates.

<Story name="Default">
  <table className="made-c-table">
    <thead>
      <tr>
        <th>Token</th>
        <th>Role</th>
        <th>Value</th>
      </tr>
    </thead>
    <tbody>
      {Object.entries(data.size.space).map(([id, token], index) => {
        return (
          <tr key={index}>
            <td>
              <code>{token.name}</code>
              <small> Themeable: {token.themeable} </small>
            </td>
            <td>{token.comment}</td>
            <td>
              <div
                className="made-s-color-block"
                style={{
                  width: `${[token.value]}`,
                  height: `${[token.value]}`,
                }}
              ></div>
              <code>{token.value}</code>
            </td>
          </tr>
        );
      })}
    </tbody>
  </table>
</Story>
