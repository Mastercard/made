# Made CSS 2.0.0 & Made Design Tokens 2.2.0

Made. June release for Design Tokens and CSS includes:

- New Toast component added to display short messages.
- New Search Input component added with squared and rounded variations.
- Notification component redesigned as well as new 'light variations' added.
- Pill component redesigned and now includes a 'branded' version.
- New responsive width utility classes added.
- New colour tokens for Action, Background, Feedback, Global, Role, Pill and Search.

#### Action Colours

Three new additional tokens for the disabled state of components. One for the Button, Toggle, and Text Input components. These tokens are for use cases such as when a Text Input needs a pre-filled disabled state, and the user must still be able to read the text.

We have also added a new focus colour token, `made-color-action-border-focus-on-dark`, to have accessible focus states for links and buttons for the new Notification and Toast components.
