# Footer

The footer is available in two variants, standard and mega.

## Default

The default footer is made up of an unordered list which is used to display plain text or links. Add the class `made-c-footer__item` to any `li` element. Links should have the class `made-c-footer__link`.

## Mega

The mega footer makes use of the Made Grid to layout elements within it and is made up of 5 distinct sections, the logo, the title, single navigation link, a group of navigation links and footer.

To add a new link to the group of navigation links insert a new `li` containing a link with the class `made-c-footer__link--navigation`.
