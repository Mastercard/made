# Typography

We have a set of type styles for headlines, eyebrow headlines and body copy to communicate visual hierarchy.

## Usage

See the Getting Started documentation to download or install the Mark for MC font. You can do a simple copy and paste, setup an automated task to move the files out of node_modules or do whatever is the best fit for your project.

## Headings

All HTML headings, `<h1>` through `<h6>`, are available.

`.made-u-h1` through `.made-u-h6` classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

Traditional headings are designed to work best in your page content. When you need a heading to stand out on a marketing-focused page, consider using a display heading — a larger, slightly more opinionated heading style.

Eyebrow text is a descriptive keyword or phrase placed above the main headline and blurb. It appears in a smaller font and sums up the content in just a few words. When users scan the headline, they'll also scan the eyebrow.

  </Story>
</Canvas>

Body text has styling by default.

For smaller body text add the `made-u-body--small` class.

For larger body text add the `made-u-body--large` class.

For helper text add the `made-u-helper-text` class.

All links can either have a Subtle or Underline style.
Subtle should be used when the placement and context of the link is explicit enough that a visible underline isn’t necessary.
Subtle links are less accessible, so don't use them for links that are critical to an experience.
Subtle links are commonly used in website footers, where there are several lists of links that are shortcuts to other pages.

| Variant   | Usage                                                                                                                                                               |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Subtle    | These links are the default link variant. They are used on their own or directly following content and they do not use underlines. They can be paired with an icon. |
| Underline | Underline links are used within a sentence or paragraph and are styled with an underline. They should NOT be paired with an icon.                                   |

For larger link text add the `made-u-body--large` class.
