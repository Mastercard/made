# Radio Button

All radio buttons must have visible labels that are associated with the radio button input.
The recommended method for most circumstances is to use the label element and an explicit association using the for and id attributes.

<Canvas>
  <Story
    name="Default"
    argTypes={{
      isChecked: {
        name: "Checked",
      },
      label: {
        name: "Label",
      },
      isDisabled: {
        name: "Disabled",
      },
      isValid: {
        name: "Valid",
      },
      validationText: {
        name: "Validation Text",
      },
    }}
    args={{
      isChecked: false,
      label: "Option 1",
      isDisabled: false,
      isValid: true,
      validationText: "Validation",
    }}
    parameters={{
      docs: {
        source: {
          code: `
<div class="made-c-form__element">
	<input class="made-c-radio" name="radio-01" type="radio" id="radio-group-1-1" value="1" />
	<label class="made-c-radio__label" for="radio-group-1-1">
		Option 1
	</label>
</div>`,
        },
      },
    }}
  >
    >{radioButton.bind({})}
  </Story>
</Canvas>

## Disabled State

Add the disabled boolean attribute on an input to give it a grayed out appearance and remove pointer events.

<Canvas>
  <Story

      },

        name: "Disabled",
      },
    }}
    args={{
      isChecked: true,
      label: "Option 1",
      isDisabled: true,
    }}
    parameters={{
      docs: {
        source: {
          code: `
<div class="made-c-form__element">
	<input class="made-c-radio" name="radio-02" type="radio" id="radio-group-1-2" value="1" checked aria-disabled="true" disabled/>
	<label class="made-c-radio__label" for="radio-group-1-2">
		Option 1
	</label>
</div>`,
        },
      },
    }}
  >
    >{radioButton.bind({})}
  </Story>
</Canvas>

## Validation - Error Messages

Add the `made-c-radio__label--error` class to the label for error validation styles.
Add an error message using the `made-c-form__validation--error` class.

<Canvas>
  <Story

    argTypes={{

      validationText: {

		Option 1
	</label>
	<div class="made-c-form__validation--error">Validation</div>
</div>`,
        },
      },
    }}
  >
    >{radioButton.bind({})}
