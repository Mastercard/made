# Checkbox

All checkboxes must have visible labels that are associated with checkbox input.
The recommended method for most circumstances is to use the label element and an explicit association using the for and id attributes.

<Canvas>
  <Story
    name="Default"
    argTypes={{
      label: {
        name: "Label",
      },
      state: {
        name: "State",
        control: {
          type: "radio",
          options: ["unchecked", "checked", "indeterminate"],
        },
      },
      isDisabled: {
        name: "Disabled",
      },
      isValid: {
        name: "Valid",
      },
      validationText: {
        name: "Validation Text",
      },
    }}
    args={{
      label: "Option 1",
      state: "unchecked",
      isDisabled: false,
      isValid: true,
      validationText: "Validation",
    }}
    parameters={{
      docs: {
        source: {
          code: `
<div class="made-c-form__element">
  <input class="made-c-checkbox" name="checkbox-01" type="checkbox" id="checkbox-group-1-1" value="1" />
  <label class="made-c-checkbox__label" for="checkbox-group-1-1">
    Option 1
  </label>
</div>`,
        },
      },
    }}
  >
    >{checkbox.bind({})}
  </Story>
</Canvas>

Add the disabled boolean attribute on an input to give it a grayed out appearance and remove pointer events.

</div>`,
