The accordion component is a list of header buttons that hide or reveal additional content when selected.

Toggle the active state classes on the title `made-c-accordion__header--active` and content `made-c-accordion__content--active` when accordion items are collapsed and expanded.

## Accessibility

If the accordion panel associated with an accordion header is visible, the header button element has aria-expanded set to true. If the panel is not visible, aria-expanded is set to false.

The accordion header button element has aria-controls set to the ID of the element containing the accordion panel content.

Optionally, each element that serves as a container for panel content has role region and aria-labelledby with a value that refers to the button that controls display of the panel.

For more information view the [W3C Accordion documentation.](https://www.w3.org/TR/wai-aria-practices-1.1/#accordion)
