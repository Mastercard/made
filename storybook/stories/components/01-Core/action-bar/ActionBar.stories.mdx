# Action Bar

The action bar is used to group together buttons with similar actions. It can contain one or more items and also has the option of adding overflow menus to create dropdown menus for individual actions. The links in the action bar all have the ARIA attribute `role="button"`.
This is because links are usually used for navigation but in this instance they are being used for actions that affect the front-end.
