# Progress Indicators

## Progress Ring

Display the progress in a circular progress bar factor, including the progress status text inside of the circle structure.

In order to set the completeness of the progress circle, the circumference of the circle must be first known. This can be calculated as follows:

_circumference = radius \* 2 \* PI_

Example, if the circle has a radius of 103, to calculate the circumference the formula would be: _103 \* 2 \* 3.14 = 647_

Once the circumference has been calculated it must be added to the values of the `stroke-dasharray` attribute of the circle element that has the class name `made-c-progress-ring__path-fill`. E.g. `stroke-dasharray="647 647"`

Once the values of `stroke-dasharray` are set to the circumference, the next step is to set how much of the circle gets filled in. This can be calculated as follows:

_circumference - percent / 100 \* circumference_

Example, using the circumference value from above of 647, in order to set the progress ring to be 75% filled the calculation would be: _647 - 75 / 100 \* 647 = 162_. Using this result set the `stroke-dashoffset` attribute of the circle element that has the class name `made-c-progress-ring__path-fill` to be 162. E.g. `stroke-dashoffset="162"`
